# Disees
 This package provides several epidemiological models (SIR, SEIR) for simulating the dynamics of infectious diseases. 
  Additionally, it includes methods for estimating the infection rate (Î²) from data and a set of regression models for analyzing risk factors and making predictions based on health-related data.
